# 🦠 COVID-19 Global Data Analysis

## 📌 Objective
Analyze global COVID-19 trends including cases, deaths, and vaccination impact across 255 countries using Python.

## 🛠️ Tools Used
- Python (Pandas, Matplotlib, Seaborn, Plotly)
- Dataset: Our World in Data

## 📊 Key Findings
- USA had the highest total cases — 10.3 crore+
- India ranked 3rd globally with 4.5 crore cases
- Yemen had the highest death rate at 18%
- Brazil had higher death rate than India despite fewer cases

## 📁 Project Structure
- `notebooks/` — Python analysis code
- `visuals/` — Charts and graphs

## 📈 Visualizations
- Line Chart — Daily cases trend (India vs USA vs Brazil)
- Bar Chart — Top 10 countries by total deaths
- Heatmap — Correlation between cases, deaths, vaccinations

## 🚀 How to Run
1. Clone the repo
2. pip install -r requirements.txt
3. Run notebooks/covid_analysis.py
